IF ELSE and SWITCH Statement Simplified

There will be a situation while coding that you might have to run a block of code which might need more than one alternative. For example, providing an alternative output if there are various inputs even though if there is only one right expected output.

In this situation, The if..else statement comes in handy which helps to create a program where there is more than one outcome.

There are 3 forms of if..else statement in Javascript.

  1. if statement
  2. if..else statement
  3. if..else if..else statement

if statement.

The if.. statement is one of the most often used statement in JavaScript. It executes a block of code or statement if the condition is evaluated to true or false(Boolean value). Even though a condition would not evaluate to a Boolean value, JavaScript will convert it into a boolean value by calling in the Boolean() function.

The statement is executed if the conditon asseses to true. On the other hand, the condition is passed on the next statment if it is assesed to false.

It is a recommended practice for a novice or new programmer to always use curly braces even if it is only one statement to be run. This helps in eliminating future confusions as it will be easy for other porgrammers to read.

You need to be carefull when running only if.. statement solely as in such conditions the code will always run even though the condition evaluates to true or false. It is best practice to run one block of code or the other and not both.

else if and else statement

In order to place and execute multiple conditions than one, we can add else..if and else statement to the if statement. This enhances the capability and application of the if statement further by providing alternatives making the code versatile to use. This helps the programmer to execute innumerable if conditions.

Nested if else statements

Nesting multiple if statements into another concept that we will explore which will help to solve more complex conditions. The use of curly braces is recommended to avoid ambiguity and to make the code clean and readable. One should not use more than 3 levels of nesting if statements and the order of the cases should be from hight expected output followed by less expected output.

Switch statement

Switch statement assess an expression and then the corresponding case body that matches the expression result is executed.

Here is the following flowchart and syntax of the switch statement:

When Switch statement is used, the variable expression is evaluated which are enclosed inside the parentheses().

  • The body used value1 is executed if the expression results matches value1.
  • The body inside value2 is executed if the expression results matches value2.
  • This process can go on depending on the number of case expressions to be evaulated.
  • If none of the case expressions match the body, then the default body is executed.

Difference between IF statement and switch statement

  • The statement to be executed will depend on the expression output inside of the if statement where as in switch statement, the user decides as to which statement is to be executed.
  • Its not easy to edit if statements if they are nested inside each other whereas switch statement is easy to edit and easily recognizable.
  • Nesting a switch inside an if..else statment is recommended compared to the other way around just to avoid th code getting too complicated.
  • if statement tests the eqality as well as logical expressions whereas switch statement tests only based on equality.
